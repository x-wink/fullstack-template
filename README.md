## 客户端

-   vite
-   vue3

# 开发准备

## 安装依赖

```shell
npm i -g pnpm
pnpm i
```

## 启动项目

```shell
pnpm run server // 启动服务端
pnpm admin // 启动管理端
pnpm client // 启动客户端
```

## 服务端调试

停掉终端命令启动的服务端，改用F5启动调试模式，即可打断点调试
